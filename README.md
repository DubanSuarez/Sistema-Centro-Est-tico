# Sistema Web para Centro Estético

Sistema web desarrollado para la gestión integral de un centro estético, permitiendo administrar pacientes, citas, servicios, especialistas e historial clínico.

## Objetivo

Optimizar la gestión operativa del centro estético mediante una plataforma que facilite el control de citas, pacientes y servicios, mejorando la organización y la atención al cliente.

## Tecnologías utilizadas

- PHP
- MySQL
- JavaScript
- HTML5
- CSS3
- Bootstrap
- FullCalendar

## Funcionalidades principales

- Gestión de citas (agenda interactiva con calendario)
- Registro y administración de pacientes
- Gestión de servicios y especialistas
- Facturación
- Historial clínico de pacientes
- Sistema de roles (administrador)

## 🖥️ Módulos del sistema

- Administrador
- Pacientes
- Especialistas
- Citas
- Facturación
- Blog (opcional)
- Usuarios
